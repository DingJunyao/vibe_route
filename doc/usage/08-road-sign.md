# 道路标志生成

本文档介绍如何使用道路标志生成工具创建符合中国标准的道路标志 SVG 图片。

## 功能说明

道路标志生成工具可以创建普通道路和高速公路的标志。

## 道路类型

### 普通道路

普通道路标志为红边白底矩形：

- **国道**：G + 三位数字（如 G221）
- **省道**：S + 三位数字（如 S221）
- **县道**：X + 三位数字（如 X221）

![普通道路标志](../img/desktop_way_pad_normal.png)

### 高速公路

高速公路标志为绿边绿底矩形：

#### 国家高速

- **G + 1-4 位数字**（如 G5、G1234）

![国家高速](../img/desktop_way_pad_expwy_national.png)

#### 省级高速

- **S + 纯数字**（如 S88）
- **S + 字母 + 可选数字**（仅限四川省，如 SA、SB、SC）

![省级高速](../img/desktop_way_pad_expwy_provincial.png)

---

## 生成步骤

### 普通道路

1. 选择道路类型：**普通道路**
2. 输入道路编号（如 `G221`、`豫S221`）
3. 点击"生成"按钮
4. 预览生成的标志
5. 点击"下载 SVG"按钮保存

![生成普通道路标志](../img/desktop_way_pad_normal_gen.png)

### 国家高速

1. 选择道路类型：**高速公路**
2. 选择子类型：**国家高速**
3. 输入道路编号（如 `G5`）
4. 可选：输入道路名称（如 "京昆高速"）
5. 点击"生成"按钮
6. 预览生成的标志
7. 点击"下载 SVG"按钮保存

![生成国家高速标志](../img/desktop_way_pad_expwy_national_gen.png)

![生成带名称的国家高速标志](../img/desktop_way_pad_expwy_national_name_gen.png)

### 省级高速

1. 选择道路类型：**高速公路**
2. 选择子类型：**省级高速**
3. 输入道路编号（如 `S88`、`川SA`）
4. 可选：输入道路名称（如 "成绵高速"）
5. 点击"生成"按钮
6. 预览生成的标志
7. 点击"下载 SVG"按钮保存

![生成省级高速标志](../img/desktop_way_pad_expwy_provincial_gen.png)

![生成带名称的省级高速标志](../img/desktop_way_pad_expwy_provincial_name_gen.png)

## 使用场景

生成的道路标志 SVG 可用于：

1. **轨迹经过区域显示**：在轨迹详情页，经过的道路会自动显示对应的道路标志
2. **视频制作**：配合覆盖层生成工具使用
3. **文档制作**：用于各类文档和报告

- 兼容性：支持所有现代浏览器和图形软件
