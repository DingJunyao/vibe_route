# 快捷操作

本文档介绍系统中的快捷操作方式。

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Esc` | 关闭对话框 |
| `Enter` | 确认表单提交 |

## 移动端手势

| 手势 | 功能 |
|------|------|
| 点击 | 选择/确认 |
| 长按 | 显示更多信息 |
| 滑动 | 滚动列表/页面 |
| 双指缩放 | 缩放地图 |

## 快速导航

### 顶部导航栏

- **Logo**：点击返回首页
- **轨迹列表**：查看所有轨迹
- **上传轨迹**：上传新的 GPX 文件
- **记录实时轨迹**：创建实时记录会话
- **道路标志**：生成道路标志 SVG（如启用）
- **后台管理**：进入后台管理（管理员）

### 移动端菜单

移动端通过右上角菜单按钮访问导航选项。
