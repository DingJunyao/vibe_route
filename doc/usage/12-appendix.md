# 附录

本文档提供系统的技术参考信息。

## 文件大小限制

| 文件类型 | 最大大小 |
|----------|----------|
| GPX 文件 | 100 MB |
| CSV/XLSX 文件 | 50 MB |
| KML/KMZ 文件 | 100 MB |

## 支持的浏览器

### 桌面端

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 移动端

- iOS Safari 14+
- Android Chrome 90+

## 坐标系说明

系统支持三种坐标系：

### WGS84

- 国际标准坐标系
- GPS 原始坐标
- 推荐使用

### GCJ02

- 中国火星坐标系
- 高德地图、腾讯地图、天地图使用

### BD09

- 百度坐标系
- 百度地图使用

上传轨迹时选择正确的原始坐标系，系统会自动转换到其他坐标系。

## 导出格式说明

### GPX 格式

- 标准 GPS 交换格式
- 支持选择导出坐标系
- 兼容大多数 GPS 设备和软件

### KML 格式

- Google Earth 格式
- 使用 Google gx:Track 扩展
- 支持导入 Google Earth 等应用

### CSV 格式

- UTF-8 编码带 BOM
- 确保 Excel 正确显示中文
- 包含所有轨迹点信息

### XLSX 格式

- Excel 表格格式
- 本系统特有格式
- 包含所有轨迹点信息

## 异步任务

对于大文件处理，系统使用异步任务：

- 地理编码填充
- 数据导入
- 覆盖层生成

任务状态可以在任务列表中查看。

## 安全说明

### 密码加密

密码使用双重加密：

1. 前端 SHA256 加密
2. 后端 bcrypt 哈希

### HTTPS

生产环境建议使用 HTTPS 确保数据传输安全。

### Token

登录后使用 JWT Token 进行身份验证，Token 有效期由后端配置。
