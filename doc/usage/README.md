## 功能概览

| 功能 | 描述 |
|------|------|
| 轨迹上传 | 支持 GPX、CSV、XLSX、KML、KMZ 格式 |
| 坐标转换 | 自动转换 WGS84、GCJ02、BD09 坐标系 |
| 地理编码 | 自动填充行政区划和道路信息 |
| 多地图支持 | 高德、百度、腾讯、天地图、OSM |
| 实时记录 | 支持 GPS Logger 实时上传 |
| 数据导出 | GPX、CSV、XLSX、KML、覆盖层图片 |
| 道路标志 | 自动生成符合中国标准的道路标志 |

## 技术要求

### 支持的部署平台

Vibe Route 支持在多种硬件架构上部署：

| 架构 | 支持状态 | 说明 |
|------|----------|------|
| x86/x64 | ✅ 完全支持 | 标准服务器和 PC |
| ARM64 | ✅ 完全支持 | 树莓派等 ARM 设备 |

**注**：ARM 架构部署需要额外的编译步骤，详见[部署指南](../deployment.md)。

### 浏览器支持

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 移动端支持

- iOS Safari 14+
- Android Chrome 90+
